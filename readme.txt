== Description ==

This simple yet usefull plugin lets the user to add custom css and js to their wordpress site.
It is tested and ready to use.

At a glance, this plugin adds the following:

* Administrators can add custom css.
* Administrators can add custom js.

A complete guide is available at https://github.com/nasiriubat/simple-custom-css-jss-wp-plugin
